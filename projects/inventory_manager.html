Inventory Manager

 A inventory managment tool made in C++ that allows you to view and edit your inventory. This project was made for CS301 Data Structures and Algorithms. The program allows the user to view their inventory, view a specific item and make changes to that item.

 The inventory is stored in a json file. To read and write json files in C++ I used this library - https://github.com/nlohmann/json. The project was kept relativly simpled and used object oriented design priciples and classes were designed using learned in the class.
